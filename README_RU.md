# Руководство по языку Вокоба (Vokoba)

**Версия: Альфа**
**Дата: Ноябрь 2025**

Вокоба — это эксперимент по перепроектированию самого механизма языка. Это априорный олигосинтетический язык, который отвергает хаос естественных языков и холодную стерильность логических в пользу новой парадигмы: прямого кодирования образов.

Вокоба не использует слова-ярлыки. Вместо этого она оперирует «понятийными атомами» — корнями, которые комбинируются для синтеза смысла так же, как мысль комбинирует образы. Это язык, где контекст не является проблемой, а становится ресурсом: грамматические маркеры явно указывают на общее знание и фокус внимания, а система финитивов интегрирует эмоциональную окраску в структуру предложения.

Цель Вокобы — стать не просто средством общения, а операционной системой для сознания, поощряющей ясность, системность и рефлексию. Данное руководство — введение в эту систему.

---

## **0. Преамбула: Условности и принципы изложения**

Данное руководство описывает структуру и использование языка Вокоба. Для ясности и удобства обучения в тексте приняты следующие условности и обозначения.

### **0.1. Условности записи**

*   **Фонетическая транскрипция:** Звучание слов и корней передаётся в квадратных скобках с использованием [Международного фонетического алфавита (IPA)](https://ru.wikipedia.org/wiki/МФА). Пример: `vokoba` — /vokoba/.
*   **Разбор слов:** Для демонстрации состава сложных слов используется дефис. Пример: `ron-i-min-a` (могущественный дух).
*   **Грамматические пометки:**
    *   `сущ.` — существительное
    *   `прил.` — прилагательное
    *   `гл.` — глагол
    *   `нар.` — наречие
    *   `част.` — частица
    *   `фин.` — финитив
*   **Опциональные элементы:** Элементы предложения, которые могут быть опущены без потери грамматической корректности, заключаются в квадратные скобки `[ ]`.
    *   Пример структуры предложения: `[Q] [T] S V O [F]`.
    *   Пример из речи: `[Tano] mika vime lyma.` — `(Сейчас)` Я вижу свет.

### **0.2. Ключевые принципы языка**

При изучении Вокобы полезно держать в уме её фундаментальные основы:

1.  **Принцип композиции:** Значение целого выводится из значения частей. Сложные идеи выражаются комбинацией простых корней.
2.  **Принцип контекстуальности:** Контекст является легитимной и управляемой частью языка. Например, маркеры вроде **`je`** (контекстный) и **`lo`** (фокус) грамматически кодируют общее для собеседников знание и акцентируют ключевые элементы высказывания. (см. п. 3.2)
3.  **Принцип эмоциональной грамматики:** Отношение говорящего к высказыванию является полноправным компонентом коммуникации и выражается с помощью **финитивов** (см. п. 6).

### **0.3. О статусе языка**

Вокоба находится на стадии **активной разработки (Альфа)**. Её словарь, корневая база и некоторые грамматические нюансы могут уточняться и расширяться. Данный документ представляет собой текущий снимок (снапшот) развивающейся системы.

Сообщества и инструменты для изучения языка находятся в процессе формирования. Настоящее руководство служит первичным источником и приглашением к соучастию в развитии проекта Вокоба.

### 2.2. Формирование составных понятий
Сложные понятия создаются путём комбинации корней двумя способами:

#### 2.2.0 Фонотактический принцип (основной закон благозвучия)

Каждое слово в Вокобе должно соответствовать чередованию гласных и согласных, выраженному в одной из двух допустимых схем:
- CVCV (согласный-гласный-согласный-гласный…)
- VCVC (гласный-согласный-гласный-согласный…)

Это правило применяется ко всему слову целиком, включая корни, связки и окончания.

Цель: избежать двух гласных подряд и труднопроизносимых согласных кластеров. 

#### 2.2.1. Простое соединение
- **Описание**: Первый корень задаёт основу, последующие уточняют её.
- **Часть речи**: Определяется окончанием (-a, -i, -e, -o, -u).

Примеры:
- *da-vok* (*da-vok-a*) — "существование + звук" → "звучание" (сущ.).
- *bon(-i)-kor* (*bon-i-kor-a*) — "хорошее + чувство" → "симпатия" (сущ.).
- *mer(-a)-pur* (*mer-a-pur-a*) — "жидкость + чистота" → "вода" (сущ.).

Если стык создаёт VV или CC, требуется вставка связующего элемента:
- между гласными → вставляется согласный -h- (фонетический буфер): da + esa → dahesa (а не daesa).
- между согласными → вставляется гласная, чтобы сохранить CVCV/VCVC. (Для более глубокого понимания какуюгласную следует использовать см. п.2.2.2)

Примеры: 
- esa + vok → esavok(e) = VCVCVC
- da + vok → davok(a) = CVCVC
- da + esa → daesa(ka) = CVVCV. Поэтому: da-h-esa → dahesa(ka).
- pon + reg → ponreg(a) = CVCCCV. Поэтому: pon-i-reg-a → ponireg(a) = CVCVCVCV. 

#### 2.2.2. Соединение с внутренним окончанием
- **Описание**: Дополнительные корни описывают качества или действия, относящиеся к последнему корню.
- **Внутреннее окончание**: Между корнями вставляется окончание части речи, определяющее связь между корнями.
- **Итоговая часть речи**: Добавляется окончание (*-a/-ka*, *-i/-ki*, *-e/-ke*, *-u/ku*), задающее роль корня в слове.

Если добавление связки образует кластер VV, используется буфер -h-: bon+i+alu → bon+i+h+alu = bonihalu(ka)

**Примеры:**
- *pon-i-reg-a* — "точечный (*pon-i*) + порядок (*reg-a*)" → "прямая" (сущ.).
- *pon-i-reg-i* — "относящийся к прямой" (прил.)
- *pon-i-reg-e* — "выпрямлять" (гл.)
- *mon-i-ob-a* — "мысляший (*mon-i*) + объект (*ob-a*)" → "мыслящий объект" (сущ.)

### 2.3. Множественное число
Множественное число образуется добавлением окончания *-r*:
- *vita* — "жизнь", *vitar* — "жизни".
- *pona* — "точка", *ponar* — "точки".
- *mika* — "я", *mikar* — "мы".

Окончание *-r* применимо ко всем частям речи (да, даже к глаголам и прилагательным).
Выражает множественность не как грамматическую категорию, а как семантическую:
- у существительных — множественность объектов
- у прилагательных и наречий — множественность проявлений качества
- у глаголов — повторяемость или распределённость действия
- у маркеров времени - неоднократность во времени (`pador` - в прошлом, завершённое, неоднократное)

### 2.4. Гендерные различия
Личные указатели нейтральны по умолчанию, но могут уточняться:
- *mika* — "я" (нейтрально), *mikar* — "мы".
- *tika* — "ты" (нейтрально), *tikar* — "вы".
- *zeka* — "оно" (нейтрально), *zekar* — "они".
- *zemaka* — "он", *zejaka* — "она".
- *mimaka* — "я" (муж.), *mijaka* — "я" (жен.).

### 2.5. Сравнительные формы
Для сравнительной формы перед основным корнем прилагательного добавляется корень "ato" (для усиления) или "nehato" со вспомогательным окончанием прилагательного "-ki" (если контекст ясен и корни сочетаются без вспомогательной гласной, то вспомогательным окончанием прилагательного можно пренебречь)
- *atokilymi*/*atolymi* — более светлый
- *nehatokilymi*/*nehatolymi* — менее светлый

Для превосходной формы перед основным корнем прилагательного добавляется корень "alu" (для усиления) или "nehalu" со вспомогательным окончанием прилагательного "-i" 
- *alukilymi* — самый светлый
- *nehalukilymi* — наименее светлый

### 2.6. Fine-tuning значений: уточнение качества/действия/состояния

В Вокобе возможно тонкое различение между абстрактным качеством и его активным проявлением. Для этого используется суффиксальный ряд `-ta/-ti/-tu/-te`, который добавляется к уже сформированному слову (с окончаниями `-a/-ka`, `-i/-ki`, `-u/-ku`, `-e/-ke`) для передачи **стиля, способа или реализованности** качества.
Этот механизм не заменяет базовые окончания, а дополняет их, позволяя выразить более тонкие оттенки смысла и часто сократить длину слова по сравнению с композицией.

#### Примеры:
- `-ta` — реализованное качество (существительное):  
  `bonu` → `bonuta` — "доброжелательность".
- `-tu` — способ действия, стиль (наречие):  
  `lib-u` → `libutu` — "беззаботным образом".
- `-te` — процесс реализации качества (глагол):
  `pur-u` → `purute` — "стремиться к чистоте/очищение".

### 2.7. Ударения

Однокорневые слова: ударение на единственном корне. 

Внутри слов:
- ударение падает на последний смысловой корень (не на связки, не на окончания).
- межкорневые связки и грамматические аффиксы не несут ударения.

В финитивах `ai` всегда ударная на a.

## 3. Грамматика

### 3.1. Структура предложения
Базовая структура: **[Q]TSVO[F]** ([вопрос] + время + субъект + глагол + объект + [финитив]).
- **Q (вопрос)**: Определяет предложение как вопросительное и является уточняющим вопросительным словом (кто?, где?, когда? и т.п.)
- **T (время)**: Указывает временной контекст и аспект. Может опускаться для аксиом или вечных истин.
- **S (субъект)**: Кто или что выполняет действие.
- **V (глагол)**: Действие или состояние.
- **O (объект)**: На что направлено действие (опционально).
- **F (финитив)**: Особая часть речи, описывающая отношение к предыдущему предложению. (опционально).

Пример: *pado mika vime sora* — "В прошлом я увидел небо".

### **3.2 Время и аспект**  

#### **Временные маркеры**  
Определяют **когда** происходит действие:  
- **pa** — прошлое.
- **ta** — настоящее.
- **fa** — будущее.
- **na** — "никогдашнее".

#### **Аспектные маркеры**  
Определяют **как** происходит действие:  
- **do** — завершённость.
- **no** — незавершённость.
- **xo** /ʃo/ — вероятность.

Примеры (не все возвожные сочетания):
- *pado* — "прошедшее завершённое",
- *tano* — "настоящее незавершённое",
- *fado* — "будущее завершённое".
- *faxo* — "будущее вероятное".
- *nado* - "никогда завершенное" (никогда не случалось)

#### **Без времени (SVO):*
- *kava kave* — "Река течёт" (вечная истина).

#### **Провторяемость:*
Если в предложении идёт речь о многократных действиях/событиях, то к маркеру времени можно добавить окончание **-r**, отмечая неоднократность.

Неоднократность может выражаться двумя способами:
— через окончание -r у временного маркера (pador) — повторение во времени;
— через окончание -r у глагола (venker) — повторение в природе действия (цикличность, прерывистость, распределённость).

#### Примеры
- mika venke do [je] — Я иду туда.
- pano mika venke [do] je — Я шёл туда.
- faxo mika venke [do] je — Я возможно пойду туда.
- pador mika venke [do] je — Я неоднократно ходил туда.
- pado mika venker [vi] je — Я похаживал там.
- nado mika venke [do] je — Я никогда туда не ходил.
- faxo mika venke [do] je — Я вероятно туда пойду.

### 3.3. Частицы/союзы
- **ni** — принадлежность: *lyma ni mika* — "мой свет".
- **vi** — внутри: *lyma vi vola* — "свет в пространстве".
- **ma** — между: mika ma lyma tu mera - "я между светом и водой"
- **ot** — от: *lyma ot mika* — "свет от меня".
- **do** — к: *lyma do tika* — "свет к тебе".
- **va** — над: *lyma va mika* — "свет надо меной".
- **neva** — под ("над" с инверсией): *mera neva mika* - "вода подо мной"
- **ta** — для: *lyma ta mika* — "свет для меня".
- **lo** — маркер фокуса: выделяет следующее за ним как центр внимания: *mika vime lo lyma* - "я вижу именно свет" (и ничто иное)
- **ri** — союз, вводящий придаточное предложение с констатацией факта ("что"): "mika vime lyma boni" просто "я вижу хороший свет", "mika vime ri luma bonu" уже "я вижу, что свет хорош"
- **je** — контекстный маркер ("тот самый"): *mona ni je* — "мысль об том (о чем мы говорим)".
- **bi** - союз, играющий роль, близкую к "но". Он противопоставояет дальнейшее высказывание предыдущему.

- **si** — "если/когда", **sida** — "тогда", **nesida** — "иначе".
- **vesi** — "если бы" (гипотетическое).
- **nesi** — "если не".
- **tu** — "и"
- **ro** — "или"

### 3.4. Отрицание и инверсия
- **ne-/neh-** — инверсия (противоположность): *neboni* — "плохой", *hehaluki* - "низкий".
- **nai-/naih-** — отрицание (отсутствие): *naiboni* — "нехороший", *naihaluki* - "невысокий".

"neh" и "naih" используются с корнями, начинающимися с гласной.

## 4. Вопросительные предложения
Структура: **Q + [T] + SVO**
- **Q** — вопросительное слово, всегда в начале.
- **[T]** — время (опционально).

### Вопросительные слова:
Вопросительные слова образуются добавлением к определяющему вопрос корню окончания -su/-asu

Некоторые вопросительные слова:
- *persu* — "кто?", *obasu* — "что?", *volasu* — "где?".
- *temasu* — "когда?", *esasu* — "как?", *genasu* — "почему?".
- *dasu* — "зачем?", *sumasu* — "сколько?", *filasu* — "чей?".

Более сложные вопросительные слова:
- *karasu* — "какой материал?", *vixasu* — "какой цвет?"

Если вопрос о множестве, то к вопросительному слову может быть добавлено множественное окончание.
- *karasur* — "какие материалы?", *vixasu* — "какие цвета?"
Примеры:
- "Karasu je oba?" — "Из чего это сделано?"
- "Karasur je oba?" — "Из каких материалов это сделано?" (подразумевается перечисление в ответе)

## 6. Финитивы

Финитивы — это особая часть речи в Вокобе, которая выражает **эмоциональное состояние или отношение говорящего** к высказыванию. Они могут находиться только **в конце предложения** и не модифицируют действие или объект, а передают настроение, с которым произнесено предложение. Финитивы опциональны и нужны для передачи настроения говорящего.

### 6.1. Окончания финитивов
- **-ai / -kai** — разговорный регистр (нейтральный, повседневный тон).
- **-afai / -fai** — официальный/вежливый регистр (официальная речь, клятвы, присяги).
- **-arai / -rai** — саркастический регистр (ироничный, насмешливый тон).
- **-amai / -mai** — поэтический регистр (возвышенный, художественный тон).

### 6.2. Правила использования
- **Позиция**: Финитив всегда стоит в конце предложения. В предложении может быть **только один финитив**.
- **Склеивание корней**: Для передачи сложного настроения можно комбинировать до двух корней. Первый корень задаёт основной тон, второй — уточняющий оттенок. Между корнями добавляется связующая гласная (*-i-*, *-a-*, *-e-*), если требуется для благозвучия.
  - Пример: *bon-i-kor-ai* — "радостно и страстно".
- **Отрицание**: Финитивы могут использоваться с префиксами *nai-* (отсутствие) или *ne-* (противоположность) для изменения настроения.
  - Пример: *nebonai* — "с огорчением", *naikorai* — "без воодушевления".
- **Вопросительные финитивы**: Финитив *resai* (или его составные формы) может использоваться для риторических или эмоциональных вопросов без начального вопросительного слова.
  - Пример: *Tika vime lyma resai?* — "Ты видел свет? Мне любопытство."
- **Эллипсис**: В диалогах финитив может заменять целое предложение, выражая настроение как самостоятельный ответ или даже целое высказывание.
  - Примеры:
    - *Je lyme?* ("То самое (о чем уже была речь) светит?") → *Alukai!* ("Говорю торжественно: да!").
	- *Resenamai?* (Как звать?) — Дословно "я спросил, поинтересовавшись именем"

### 6.3. Некоторые примеры финитивов
| Корень                   | Финитив  | Отношение говорящего          | Пример предложения                                         |
|--------------------------|----------|-------------------------------|------------------------------------------------------------|
| **kor** (эмоция)         | *korai*  | страстно, с воодушевлением    | *Mika vime lyma korai!* — "Я видел свет! (я воодушевлён)"  |
| **suf** (неясность)      | *sufai*  | неуверенно, сомнительно       | *Zeka monapake sufai...* — "Он знает... (я в сомнениях)"   |
| **bon** (добро)          | *bonai*  | радостно, одобрительно        | *Tika zamke bonai!* — "Ты сделал это! (я рад)"             |
| **alu** (высота)         | *alukai* | торжественно, возвышенно      | *Lyma lyme alukai!* — "Свет сияет! (торжественно)"         |
| **lib** (свобода)        | *libai*  | легко, беззаботно             | *Venke libai!* — "Идём! (я беззаботен)"                    |
| **res** (вопрос)         | *resai*  | вопросительно, с любопытством | *Tika vime lyma resai?* — "Ты видел свет? (мне любопытно)" |
| **zel** (мечта)          | *zelai*  | мечтательно, с надеждой       | *Tano mika venke zelai.* — "Сейчас я иду. (с надеждой)"    |
| **san** (здоровье)       | *sanai*  | заботливо                     | *Tika zamke sanai!* — "Ты сделал это! (с заботой)"         |
| **pul** (взаимодействие) | *pulai*  | дружелюбно, открыто           | *Tika venke pulai!* — "Ты идёшь. (дружелюбно)"             |

### 6.4. Составные финитивы
Составные финитивы комбинируют два корня для передачи сложного настроения:
- *bon-i-kor-ai* — "радостно и страстно": *Pado tika zamke bonikorai!* — "Ты сделал это! (говорю это с радостью и страстью)"
- *suf-i-res-ai* — "неуверенно и вопросительно": *Pado zeka vime lyma sufiresai?* — "Он видел свет? (говорю это с сомнением)"
- *zel-i-lib-ai* — "мечтательно и беззаботно": *Mika venke zelilibai!* — "Я иду. (говорю это мечтательно и беззаботно)"

### 6.5. Регистры в действии
- Разговорный (*-ai/-kai*): *Mika vime lyma korai!* — "Я видел свет! (говорю с воодушевлением)"
- Официальный (*-afai/-fai*): *Lyma lyme alufai!* — "Свет сияет! (говою торжественно, официально-вежливо)"
- Саркастичный (*-arai/-rai*): *Mika vime lyma korarai!* — "Я видел свет! (говорю с воодушевлением (на самом деле нет))"

### 6.6. Модификация действия
К финитиву можно приставить ещё один корень, который будет определять форму высказывания.
Примеры:
- Tika zamke je resevitai? - Ты сделал это? — Спрашиваю (не просто говорю) энергично/настойчиво. — res-e - вопрос-глагол
- Tika zamke je ronehakokai! - Ты сделаешь это. — Приказываю! — ron-e - власть-глагол, ako - изменение, развитие

### 6.7. Примеры диалогов
1. **Простой диалог**:
   - A: *pado tika vime zeka?* — "Ты видел его?"
   - B: *korai!* — "[Да!] (с воодушевлением)"
   - A: *temasu?* — "Когда?"
   - B: *nemikoregi lymatema zelai!* — "Вчера. (с надеждой)" ("nemikoregi lymatema" здесь в контексте дней инвертирует "первый день" (mikoregi lymatema) в "первый день в прошлом")

2. **Саркастический ответ**:
   - A: *pado tika zamke?* — "Ты сделал?"
   - B: *bonarai!* — "Ну да, конечно! (сарказм)"

3. **Риторический вопрос**:
   - *zeka monapake do je sufiresai?* — "Он знает это? (я спрашиваю с сомнением и любопытством)"

### 6.8. Примечания
- Финитивы не заменяют наречия, которые описывают способ действия. Финитивы выражают только отношение говорящего к высказыванию.
  - Наречия (-u/-ku) описывают как происходит действие.
  - Финитивы (-ai и прочие) выражают отношение к действию.

**Пример критического различия наречия и финитива**
- *Наречие* — tano tika vene bonu → "Сейчас ты идёшь радостно"
- *Финитив* — tano tika vene bonai → "Сейчас ты идёшь (говорю это радостно)"

## 7. Примеры предложений
- *pado mika vime lyma* — "В прошлом я увидел свет".
- *tano mika monapake do tika* — "Сейчас я знаю тебя".
- *si tika dapake mona do je, sida esavoke* — "Если ты знаешь это, говори".

## 8. Система именования чисел в Vokoba

### **8.1. Основные числа**

Vokoba использует систему счисления, основанную на группировке и разрядах. Каждое число представлено комбинацией цифр и разрядов. Числа формируются с использованием определённых префиксов и суффиксов, следуя регулярной структуре.

#### **8.1.1. Основные цифры**

- **neko** — 0 (не цифра)
- **miko** — 1 (моя цифра)
- **tiko** — 2 (твоя цифра)
- **zeko** — 3 (их цифра)
- **sano** — 4 (здоровая цифра)
- **tago** — 5 (сегментированная цифра)
- **rego** — 6 (упорядоченная цифра)
- **libo** — 7 (свободная цифра)
- **mero** — 8 (жидкая цифра)
- **aluko** — 9 (верхняя цифра)

Названия в скобках добавлены для удобства запоминания.

Эти цифры используются для формирования более крупных чисел, когда они комбинируются с маркерами разряда.

#### **8.1.2. Количественные слова**

- **ronatiko** — десяток (ron (мощь, сила) + два)
- **ronazeko** — сотня (ron (мощь, сила) + три)

### **8.2. Единицы измерения разряда**

Концепция **разрядов** позволяет определять величины, такие как десятки, сотни, тысячи, миллионы и т. д. Каждый разряд имеет уникальный префикс:

**-volo/-volor** — разряд для больших чисел (миллионы, миллиарды и т. д.)

- **tivolo** — тысяча (10^3)
- **zevolo** — миллион (10^6)
- **sanvolo** — миллиард (10^9)

### **8.3. Группировка чисел**

Числа группируются в зависимости от их разряда и соответствующей величины:

#### **8.3.1. Формирование больших чисел**

Каждая группа представляет собой отдельный разряд, и число в каждой группе формируется с использованием соответствующей комбинации цифры и названия разряда:

- **1,000,000** = **[miko] zevolo**
- **1,000** = **[miko] tivolo**
- **123,000,000,000** = **miko ronazeko tiko ronatikor zeko sanvolor**

#### **8.3.2. Сочетание разрядов и чисел**

Число **123,456,789** будет записано как:

```
[miko] ronazeko tiko ronatikor zeko zevolor, sano ronazekor tago ronatikor rego tivolor, libo ronazekor mero ronatikor aluko.
```

Здесь:
- **[miko] ronazeko** = [одна] сотня
- **tiko ronatikor** = два десятка
- **zeko** = три
- **zevolor** = миллионов
- **sano ronazekor** = четыре сотни
- **tago ronatikor** = пять десятков
- **rego** = шесть
- **tivolor** = тысяч
- **libo ronazekor** = семь сотен
- **mero ronatikor** = восемь десятков
- **aluko** = девять

Или можно использовать сокращённую запись:
```
mi-ti-ze-san-tag-reg-lib-mer-aluko
```

Или скомбинировать подходы
```
mi-ti-zeko zevolor san-tag-rego tivolor lib-mer-aluko
```

### **8.4. Особые случаи**

- **Ноль в разрядах:**  
Ноль может быть опущен в тех разрядах, где нет других чисел.  
Либо, если это необходимо, можно уточнить:  
-  **naironatiko** - нет дюжины  
-  **naironazeko** - нет сотни  
-  **naitivolo** - нет тысячи  
-  **naizevolo** - нет миллиона  
и так далее.

### **8.5. Десятичные дроби**

Используйте `-ne-` для отрицательных степеней.

- 0.1 = [miko] ronanetiko (ron + ne + ti + ko) = [одна] десятая.
- 0.01 = [miko] ronanezeko (ron + ne + ze + ko) = [одна] сотая.
- 0.001 = [miko] ronanesano (ron + ne + san + o) = [одна] тысячная
- 0.03 = zeko ronanezekor = три сотых

Для более сложных чисел можно использовать выражение `neko pona`, что означает «ноль точка...», а затем использовать основную систему.

123.456 = miko ronazeko tiko ronatikor zeko *pona* sano ronazekor tago ronatikor rego

### **8.6. Дробные числа**

В вокобе **vi** означает внутри чего-то. Например `lyma vi mika` — "свет внутри меня".

Используйте это для указания числа **внутри** дробного целого.
- 1/3 = miko vi zekor = один внутри трёх
- 2/5 = tiko vi tagor = два внутри пяти
- 21/13 = tiko ronatikor miko vi [miko] ronatiko zekor = два полных десятка и единица внутри [одного] десятка и трёх (можно опустить `miko` здесь)

Обратите внимание на множественное число у знаменателя. Это значит, что третьих и пятых в целом много, а именно три и пять.

### **8.7. Порядковые числительные м особые сочетания**

- mikolimi — единственный, только один. (miko-lim-i) Буквально "один-ограничение-прилагательное"
- tikoregi — второй по порядку. (tiko-reg-i) Буквально "два-порядок-прилагательное"
- mikoponu — во-первых (miko-pon-u)
